# Jogo da Forca - JavaScript

Projeto desenvolvido para a disciplina de Programação.

## Descrição

Este projeto consiste em uma implementação do jogo da forca utilizando HTML, CSS e JavaScript.

Toda a lógica do jogo é executada diretamente no navegador, sem necessidade de servidor ou banco de dados.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
- Git
- GitHub

## Funcionalidades

- Palavra secreta aleatória
- Controle de tentativas
- Registro de letras digitadas
- Mensagem de vitória
- Mensagem de derrota
- Interface estilizada
